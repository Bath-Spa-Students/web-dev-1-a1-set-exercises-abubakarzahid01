CSS tages Examples

The following are examples of different css tags:

Color Property:

Color Property:
            p {
                color: blue;
            }
            <p>This CSS rule sets the text color of all 
 elements to blue.</p>

Font Size Property

Font Size Property:
        h1 {
            font-size: 24px;
        }
        <p>This CSS rule sets the font size of all elements to 24 pixels.</p>

Margin Property

Margin Property:
            .container {
                margin: 20px;
            }
            <p>This CSS rule sets a margin of 20 pixels on all sides of elements with the class .container.</p>
